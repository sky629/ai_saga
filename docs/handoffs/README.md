# Handoff Documents

Store task-specific handoff documents in this directory.

## Usage

- Copy `docs/HANDOFF_TEMPLATE.md`
- Create a new file in this directory
- Fill it for each phase-thread or sub-agent handoff

## Recommended Naming

- `YYYY-MM-DD-<topic>-phase.md`
- `YYYY-MM-DD-<topic>-task.md`
- `YYYY-MM-DD-<topic>-review.md`
- `YYYY-MM-DD-<topic>-qa.md`
- `YYYY-MM-DD-<topic>-integration.md`

## Example

- `2026-03-17-combat-response-phase.md`
- `2026-03-17-combat-response-task.md`

## Rules

- One handoff document per meaningful transfer of ownership
- Update the same file while the same handoff is active
- Create a new file when ownership or phase changes
