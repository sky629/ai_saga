# Handoff Documents

이 디렉터리는 현재 진행 중이거나 가장 최근에 완료된 작업의 handoff 문서를 보관한다.

## Current Policy

- 최신 handoff 문서만 유지한다.
- 오래된 handoff 문서는 누적 보관하지 않는다.
- `README.md`와 최신 handoff 1개만 남기는 것을 기본 규칙으로 한다.

## Usage

- 새로 큰 작업을 시작하거나 phase/ownership가 바뀌면 `docs/HANDOFF_TEMPLATE.md`를 기준으로 새 handoff를 만든다.
- 같은 작업을 이어가는 동안에는 같은 파일을 계속 업데이트한다.
- 작업이 끝나고 새 최신 handoff가 생기면 이전 handoff들은 정리한다.

## Naming

- `YYYYMMDD-<topic>.md`
